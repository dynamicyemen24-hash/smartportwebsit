4. **تحديث `.vercelignore`** - تم تنظيفه لضمان عدم رفع الملفات غير الضرورية

## 🚀 خطوات النشر على Vercel

### الطريقة 1: النشر عبر GitHub (موصى به)

1. **ارفع المشروع إلى GitHub:**
   ```bash
   git init
   git add .
   git commit -m "Smart Ports Website - جاهز للنشر"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/smartports-website.git
   git push -u origin main
   ```

2. **اربط Vercel بـ GitHub:**
   - اذهب إلى [Vercel Dashboard](https://vercel.com/dashboard)
   - انقر على **Add New... > Project**
   - اختر **Import Git Repository**
   - ابحث عن `smartports-website` واختره
   - انقر **Import**

3. **أضف متغيرات البيئة في Vercel:**

   ```
   VITE_SUPABASE_PROJECT_ID=tnzlusiymgdvsqfufmme

   VITE_SUPABASE_ANON_KEY=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJpc3MiOiJzdXBhYmFzZSIsInJlZiI6InRuemx1c2l5bWdkdnNxZnVmbW1lIiwicm9sZSI6ImFub24iLCJpYXQiOjE3NzgxNDM5MTgsImV4cCI6MjA5MzcxOTkxOH0.cFK0ewweyiP63-h4v1cJgZaNhBkcK37bU8mptfggNNw
   ```

```bash

# سجل الدخول

# انشر المشروع
vercel --prod
```

## 🖥️ التشغيل المحلي

لتشغيل الموقع محلياً:

```bash
# تثبيت الاعتمادات
pnpm install

# تشغيل وضع التطوير
pnpm dev

# أو بناء وعرض النسخة الإنتاجية
pnpm build
pnpm preview
```

الموقع سيعمل على: `http://localhost:5173` (للتطوير) أو `http://localhost:4173` (للنسخة الإنتاجية)
